# Superpowers Session Start

At the start of a Cline session, apply the Superpowers bootstrap rule:

1. Check whether a Superpowers skill applies before doing work.
2. Load the matching skill from `.agents/skills/<skill-name>/SKILL.md`.
3. Follow user instructions first when they conflict with a skill.
4. Use Cline-native tools for file reads, edits, commands, checklists, and delegation.

This hook is intentionally lightweight. The actual process instructions live in the installed skills.
